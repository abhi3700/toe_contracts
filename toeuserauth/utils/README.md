## Calculate Profile Hash
Here, the profile hash is calculated for set of personal information fields:

* name
* address
* national_id

## Compile
* `$ mkdir build && cd build`
* `$ cmake ..`
* `$ sudo make` or `$ make`
* `$ ./profile_hash`
